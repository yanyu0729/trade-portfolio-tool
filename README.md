# Trade Portfolio Tool

This repository contains the spreadsheet-based tool accompanying the article:

**Wang, Y., Gu, B., & Zhang, X. (2026).**
*Shifting U.S.-China trade from feed to food reduces global agricultural nitrogen loss and greenhouse gas emissions.*
**Nitrogen Cycling.**

## Overview

The spreadsheet-based tool allows users to explore the environmental impacts of partial transitions in U.S.–China agricultural trade portfolios.

Users can modify parameters including:

- Percentage of imported feed crops replaced by animal products
- Crop nitrogen use efficiency (NUE)
- Livestock nitrogen use efficiency (NUE)
- Manure recovery rate 

The tool estimates resulting changes in:

- Animal N products
- Nitrogen losses
- Greenhouse gas emissions
